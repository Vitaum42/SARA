# 🏛 SARA — Sistema Para Acompanhamento Regional de Atuação Política

O **SARA** é um sistema web para gestão, monitoramento e análise da atuação política em nível regional, centralizando dados de políticos, gastos públicos, demandas da população e desempenho legislativo.

---



- 📊 **Dashboard** com métricas gerais do sistema  
- 👤 **Gestão de políticos** (cadastro, listagem e filtros)  
- 💰 **Controle de gastos públicos**  
- ✅ **Verificação de elegibilidade** (integração com API externa)  
- 📋 **Registro e acompanhamento de demandas públicas**  
- 🏛️ **Atuação legislativa** (proposições, votações, discursos, frentes)  
- 📑 **Geração de relatórios**
